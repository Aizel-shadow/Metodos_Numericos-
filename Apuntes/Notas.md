## EJERCICIO 1 
La sumatoria $1 + 1/2 +1/4 ...$ tal que el error absoluto $e_{abs} < 10^{-1}$. 

```python
suma = 0
n = 0
while abs(2 - suma) >= 0.1:
    suma += (1/2)**n
    n += 1
print(f"terminos: {n}, Suma: {suma:.6f}, Error: {abs(2-suma):.6f}")
```
